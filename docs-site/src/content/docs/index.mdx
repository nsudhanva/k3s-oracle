## Features

<CardGrid stagger>
	<Card title="Always Free" icon="star">
		Runs entirely on OCI's Always Free tier using Ampere A1 Compute (4 OCPUs, 24GB RAM).
	</Card>
	<Card title="Automated NAT" icon="pencil">
		Solves OCI's lack of free NAT Gateway with a software router on the Ingress node.
	</Card>
	<Card title="GitOps Driven" icon="add-document">
		Managed by Argo CD with the App-of-Apps pattern. Infrastructure as Code via Terraform.
	</Card>
	<Card title="Gateway API" icon="setting">
		Exposes services via Kubernetes Gateway API (Traefik) on ports 80/443 without paid Load Balancers.
	</Card>
</CardGrid>
